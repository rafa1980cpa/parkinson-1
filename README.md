# NeuroTempo PRO

Ecosistema digital de rehabilitación para pacientes con Parkinson.

**Funcionalidades:**
- Terapia de Ritmo RAS con metrónomo Web Audio API
- Análisis de informes médicos con OCR + NLP clínico
- Test de Tapping, Vocal, Dibujo y Respiración
- SOS de emergencia y contactos configurables
- Detección de congelación de marcha (FOG)
- Bilingüe: Español / English
